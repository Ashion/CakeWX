# CakeWX

**CakeWX** 是一个由PHP写的的免费的开源微信公众号管理平,遵循 [MIT License](https://github.com/niancode/CakeWX/blob/master/LICENSE).

他是基于 [CakePHP](http://www.cakephp.org) MVC 框架.

## Requirements
  * Apache with `mod_rewrite`
  * PHP 5.2 or higher
  * MySQL 4.1 or higher
